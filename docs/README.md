# Flow-Tracker 🌊

> Tracker de hàbits i xarxa social · DAM AMS2 · MP13

Aplicació multiplataforma per registrar hàbits diaris i compartir el progrés amb amics.

## Stack tecnològic

| Capa       | Tecnologia                    |
|------------|-------------------------------|
| Frontend   | Flutter (Android, Web, Windows, Linux) |
| Backend    | Node.js + Express             |
| Base de dades | PostgreSQL + Prisma ORM    |
| Desplegament | Proxmox + Nginx + PM2       |

## Estructura del projecte
```
flow-tracker/
├── backend/    # API REST (Node.js)
└── frontend/   # App Flutter
```
